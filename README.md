# 📚 Ingeniería II - 2° Año

Bienvenido a este repositorio correspondiente a la materia **Ingeniería II**, del segundo año de la carrera **Tecnicatura Superior en Desarrollo de Software**.

---

## 🎯 Objetivos de la Materia

- Aplicar conceptos de desarrollo de software en proyectos prácticos.
- Comprender y utilizar tecnologías web modernas.
- Fomentar buenas prácticas de programación y organización de código.
- Trabajar de manera colaborativa en el desarrollo de aplicaciones.

---

## 🛠️ Tecnologías Utilizadas

En este repositorio se trabajará con las siguientes tecnologías:

- **HTML5** → Estructura de las páginas web  
- **CSS3** → Estilos y diseño visual  
- **JavaScript** → Lógica y comportamiento dinámico  
- **Bootstrap** → Framework para diseño responsive y componentes UI
