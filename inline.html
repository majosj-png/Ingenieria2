Demostración de etiquetas inline

Este documento muestra varias etiquetas inline en HTML:

Texto normal con un span, un énfasis (em), un fuerte (strong), un subrayado (u), un itálica (i) y un texto pequeño.

Ejemplo de código inline: let x = 10; y una porción marcada: resaltar.

Abreviatura: HTML — fecha/tiempo: 2026-04-23.

Superíndice y subíndice: E = mc2, H2O.

Dirección/aislamiento: texto con bdo, texto aislado.

También hay enlaces inline: visita Example.
